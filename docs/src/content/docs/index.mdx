## What is Clawbolt?

Clawbolt is a messaging-first AI assistant that helps solo tradespeople manage their business. Text your assistant from iMessage, RCS, SMS, or Telegram. No app to install, no dashboard to learn. Just text.

Built by [Mozilla.ai](https://mozilla.ai) using the open-core model.
